What are Variables in C Language?

A variable in C is a specifically defined area of memory where a value is stored. These values can be of several data types, including characters, integers, and floating-point numbers. The name of the variable indicates the type of data it stores and its value may change during the execution of the program.

Let's explore variables:

  1. Declaring a Variable: Declaring a variable is a process used to specify the name and data type of the variable.
    2. int num;
  2. Initializing a Variable: Initializing a variable means giving a starting value to the variable.
    3. num = 0;
  3. Assigning a Variable: Assigning a variable means giving a new value to the variable.
    4. num = 10;

Example: Variables

        #include <stdio.h>
        int main() {
            // Variable Declaration
            int num;

            // Variable Initialization
            num = 0;
                
            // Variable Assignment
            num = 10;
                
            // Printing the value of variable
            printf("Value of num: %d\n", num);
                
            return 0;
        }

Explanation about Program:

  • First, we declare an integer variable "int num".
  • Second, we initialize "num" with 0: "num = 0".
  • Third, we assign "num" with 10: "num = 10".
  • Lastly, we print the value of the variable "num" using printf.
