Chapter.6.3

One Dimensional Array

Once an array is defined, each element of the array can be accessed by the array name followed by an index enclosed in brackets. All array elements are numbered starting from 0.

Example:

int s[5] = {10, 20, 30, 40, 50};

So the valid range of subscripts is 0 to 4, i.e., the lower bound is 0 and the upper bound is one less than the specified value in the subscript, i.e., 5-1 = 4.

Two Dimensional Array

In a two-dimensional array, an element can be accessed simply by the array name followed by two index numbers enclosed in brackets.

Example:

int a[2][3] = {10, 20, 30, 40, 50, 60};
Access Array Element Value
To access the first row and first column a[0][0] 10
To access the first row and second column a[0][1] 20
To access the first row and third column a[0][2] 30
To access the second row and first column a[1][0] 40
To access the second row and second column a[1][1] 50
To access the second row and third column a[1][2] 60
