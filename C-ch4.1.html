Chapter.4.1

Types of Functions

Basically, there are two types of functions. They are:

  • Library Functions
  • User Defined Functions

1. Library Functions

There are a number of library functions available in C that are used to carry out various operations and calculations. These are predefined in the compiler of the language and stored in a special library file.

If a programmer requires a particular library function to be used in a program, the corresponding library file has to be included in the program, e.g., #include<math.h> for mathematical functions. The corresponding library function is accessed and is attached to the program during the compilation process.

2. User Defined Functions

A function is a self-contained block of statements that carries out some specific, well-defined task specified by the user and is called a user-defined function or simply a function.

Each and every C program always consists of one or more than one functions. One of these must be main(), from which the execution of the program begins.

There is no function that can work independently. They have to be called within the main() function.
