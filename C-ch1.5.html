Chapter.1.5

What is the Constants ???

Constants, The name clear that it has fixed value that remain the same while a program is executed. They are used to represent the mathematical constants, configuration parameters and more that remains constant during the execution of the Program.

Define and Initialize Constants

First, we want to define constants, then we can easily use them. To define a constant, we use the const keyword. 

// Constant Definition
const float PI = 3.14;
const int MAX_VALUE = 100;

Here we declare and initialize PI by writing const float PI = 3.14;. We also declare and initialize MAX_VALUE by writing const int MAX_VALUE = 100;.

Using Constants

Once constants are defined, they can be used in your code wherever you would use the corresponding literal values. For example:

// Using Constants
float radius = 5.0;
float area = PI * radius * radius;

Here we declare a variable radius and assign the value 5.0. Then we calculate area using the constant PI and radius.

Example: Constants

Here is an example to understand how the code example works for constants:

#include <stdio.h>
int main() {
    // Constant Definition
    const float PI = 3.14;
    const int MAX_VALUE = 100;

    // Using Constants
    float radius = 5.0;
    float area = PI * radius * radius;

    printf("Value of PI: %f\n", PI);
    printf("Maximum Value: %d\n", MAX_VALUE);
    printf("Area of Circle with radius %.2f: 
    %f\n", radius, area);

    return 0;
}
