Chapter.1.4

What are Data Types?

Data Types are like blueprints, which tell us how much space a variable requires and which values are stored in it. In other words, Data Types are used in C Programming to specify the kind of data that a variable can store. They determine how much memory is used or allocated to a variable and what kind of values it can hold.

Types of Data Types

Here we will study about three major Data Types which are mostly used:

  1. Integer Data Type
  2. Character Data Type
  3. Floating-Point Data Types

1. Integer Data Type

Here we have Integer Data Type, which allows us to store whole numbers. Like Integer – "int" takes 4 bytes of memory space which is responsible for storing the whole number. Let's take an example:

// Integer Data Type
int num = 10;

Here we write "int num = 10;" – Declare and Initialize an Integer variable "num".

2. Character Data Type

Character Data Type gives permission to store characters. Like Character “char” takes 1 byte of memory space. Which is used to store a single character. Let's take an example:

// Character Data Type
char letter = 'A';

Here we write "char letter = 'A';" – Declare and Initialize a character variable "letter".

3. Floating-Point Data Type

Floating-Point Data Type gives permission to store decimal point values. Like Floating-Point “float” takes 4 bytes of memory space. Which is used to store Floating-Point Values.

// Floating-Point Data Type
float decimal_num = 3.14;

Here we write "float decimal_num = 3.14;" – Declare and Initialize a Floating-Point variable "decimal_num".

Example: Data Types

Here is an example of Data Types in C Language:

#include <stdio.h>
int main() {
    // Integer Data Type
    int num = 10;

    // Floating-Point Data Type
    float decimal_num = 3.14;

    // Character Data Type
    char letter = 'A';

    // Printing the values of variables
    printf("Integer Number: %d\n", num);

    printf("Floating-Point Number: %f\n", 
    decimal_num);
    
    printf("Character: %c\n", letter);

    return 0;
}
