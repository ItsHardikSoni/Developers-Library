Chapter.2 - Conditional Control Statements

Introduction

Conditional Control Structures allow us to make decisions based on certain conditions in our program. In simple terms, they help us to execute different blocks based on whether a condition is false or true.

Why do we use Conditional Control Structures?

Conditional Control Structures are basically used to make decisions in our program. Just take an example: we have an apple. Then we write two conditions on it. "Condition 1" is for if I eat the apple, then it will finish. "Condition 2" is for if I do not eat it, then it will remain the same. If I set the logic for eating the apple, it will show the output of "Condition 1".

Like this real-world example, we also set these conditional logics in our system programming or when creating software.

Types of Conditional Control Structures

In Conditional Control Structures, we have mainly four types of statements which are helpful in creating decisions in our programs:

  • If Statement
  • If-else Statement
  • Nested If-else Statement
  • Switch Case Statement
