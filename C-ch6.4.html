Chapter.6.4

Introduction

As data can be entered into an Array by using loop statements, data can also be traversed by using loop statements.

To traverse each element of an array, we use a variable in the subscript part in place of a constant (for a single element).

Example: One Dimensional Array

// Demonstration of Outputting One Dimensional Array
        printf("\n Elements of an array are\n");
        for (i = 0; i < 9; i++)
            printf("%d \t", S[i]);

Example: Two Dimensional Array

// Demonstration of Outputting Two Dimensional Array
        printf("\n Elements of an array are\n");
        for (i = 0; i <= 4; i++) {
            for (j = 0; j <= 5; j++)
                printf("%d\t", a[i][j]);
            printf("\n");
        }
