Chapter.1.6

What are Input/Output Functions?

Input/Output Functions allow us to communicate with the user. Input Functions allow us to take information from the user, and Output Functions allow us to display that information which we collect from the user.

Input/Output Functions

We have printf() and scanf(), which are used to display output to the user and take input from the user, respectively. Let’s talk about both and understand briefly how they work.

printf() Function

printf() – This function is used to display the output on the screen. It allows you to format and print various types of data, such as text and variables. For example, if you want to print "Hello World", you would write:

printf("Hello, World!\n");

scanf() Function

scanf() – This function is used to take input from the user. For example, if you want to take a number from the user, you would write:

scanf("%d", &num);

%d means we are expecting an integer value and & is used to indicate the address of the variable. This is because scanf() requires the address of the variable to store the input.

Example: Input/Output

Let’s take an example of input/output functions to understand them briefly:

#include <stdio.h>
        int main() {
            int num;
            
            // Input
            printf("Enter a number: ");
            scanf("%d", &num);
            
            // Output
            printf("You entered: %d\n", num);
            
            return 0;
        }
