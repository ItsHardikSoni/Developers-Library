What are the Switch Case Statement ???

Switch Case is a Conditional Control Structure in C Language, which is used to check the value of one variable and execute code blocks accordingly. In other words, it is especially helpful when you want to compare the value of a single variable or expression to the values of many other constants.

Syntax:

Here is the syntax of the Switch Case Statement which shows how the switch case works,


switch (variable) {
    case value1:
        // code block to execute if variable equals value1
        break;
    case value2:
        // code block to execute if variable equals value2
        break;
    ...
    default:
        // code block to execute if variable doesn't match any case
}

Here, “variable” is the variable whose value is being checked, and “value1”, “value2”, etc., are the possible values it can take.

Example:

Here is the example of Switch Case Statement,


#include <stdio.h>
int main() {
    char grade = 'B';
    // Switch case statement example
    switch (grade) {
        case 'A':
            printf("Excellent!\n");
            break;
        case 'B':
            printf("Good!\n");
            break;
        case 'C':
            printf("Average!\n");
            break;
        default:
            printf("Need Improvement!\n");
    }
    return 0;
}

Explanation about Example:

Here we use the Switch Case Statement to check the performance of a student based on the “grade” variable. We define different cases, and according to the true value, the message will be printed, which we set in our cases.
