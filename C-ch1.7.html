Chapter.1.7

What are Operators?

Operators are symbols which are used to perform operations on operands. In simple words, they are like tools which help to manipulate data in our program. Operators are used to perform mathematical calculations such as adding, subtracting, etc.

What are Expressions?

A set of values, variables, and operators combined to produce a single value is called an Expression. Expressions can range in complexity from a single value to a combination of several processes.

Types of Operators

In C Language, we have five types of operators:

  • Arithmetic Operator
  • Relational Operator
  • Logical Operator
  • Assignment Operator
  • Increment and Decrement Operator

1. Arithmetic Operator

Arithmetic operators are mainly used for mathematical operations like addition, subtraction, etc.

Name of Operator Symbol of Operator
Addition +
Subtraction -
Multiplication *
Division /
Modulus %

2. Relational Operator

Relational operators are used to compare values, such as less than, greater than, etc.

Name of Operator Symbol of Operator
Equal to ==
Not Equal to !=
Less Than <
Greater Than >
Less than or Equal to <=
Greater than or Equal to >=

3. Logical Operator

Logical operators are used to perform logical operations.

Name of Operator Symbol of Operator
Logical AND &&
Logical OR ||
Logical NOT !

4. Assignment Operator

Assignment operators help us to assign values in our program.

Name of Operator Symbol of Operator
Assign =
Add and Assign +=
Subtract and Assign -=
Multiply and Assign *=
Divide and Assign /=
Modulus and Assign %=

5. Unary Operator

Unary operators act on a single operand.

Name of Operator Symbol of Operator
Increment ++
Decrement --
Negation -
Logical NOT !

Example: Operators and Expression

Let’s write a program where we write expressions using operators:

#include <stdio.h>
int main() {
    // Arithmetic Operators
    int num1 = 10, num2 = 5;
    int sum = num1 + num2;
    int difference = num1 - num2;
    int product = num1 * num2;
    // Typecasting to get accurate division
    float quotient = (float)num1 / num2;
    int remainder = num1 % num2;
    
    // Output
    printf("Sum: %d\n", sum);
    printf("Difference: %d\n", difference);
    printf("Product: %d\n", product);
    printf("Quotient: %.2f\n", quotient);
    printf("Remainder: %d\n", remainder);
    
    return 0;
}

In this program, first we declare variables, then we assign values to those variables, and finally we create expressions using the operators.
